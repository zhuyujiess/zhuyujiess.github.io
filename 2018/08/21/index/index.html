仲夏夜之星

Posted by 朱玉洁 on 2018-08-21
前端 JavaScript Angular

最近手机依赖症好像是愈发严重，总是在还有几分睡意的时候，睁着朦胧的双眼在床上摸索手机。拿到手机后，习惯性地刷刷微博、逛逛脸书，然后扭头继续睡觉。今天微博推送的一条关于“放羊的星星播出十周年”纪念日，让以前的记忆扑面而来，立时睡意全无。
微博真的是挺讨厌，总是会给你推荐一些内容，告诉你你已经不再年轻，你已经是一个有故事的老同学。回忆很长并且是一个漫无边际的混沌，所以我们的大脑就以时间节点的形式把我们的过去连接起来，以后回忆的时候就找到对应的时间节点蔓延开来。在我的记忆中，大到非典、载人飞船升空、成功举办奥运会，小到一部电影的上映，一首新歌的发布或是近期NBA总决赛的进行，每个时间点发散开来的记忆都能被写成一部回忆满满的书。

微博推送的关于放羊的星星这部电视剧，便是我读书时代很爱看的偶像剧（^^那个时候男孩子看偶像剧应该不会被鄙视吧）。时光荏苒，转眼之间又是十年以后，到现在我依然记得剧中浪漫唯美的手链——仲夏夜之星。以下是这条手链的简介：
仲夏夜之星是电视剧《放羊的星星》中夏之星为她和仲天骐设计的手链，Somnus (仲夏夜之星)
是拉丁语中罂粟花的学名
也是罗马众神中的睡眠之神——索莫纳斯。
Somnus 含义：
美丽却致命
一旦迷上了一开始会很快乐
后来就会付出惨痛的代价
这不正是有点像爱情的本质。
(^^现在看起来是不是很幼稚)
也就是因为那部电视剧，我开始喜欢林志颖这个明星，并且把他的照片作为我的QQ图像接近十年之久。

年纪小的好处就是没有什么需要回忆的，有的只是对未来的憧憬和希望。那个时候，老师会让我们写作文，写我们的梦想和若干年后的自己，然后去讲台朗读给全班同学听。当时课堂活泼热闹，大家争恐后，科学家、警察、医生，一个个梦想说的铿锵有力、笃信不疑。然而时间总是稳步向前，不管你愿不愿意、有没有准备好，它总是分秒不差地把你带到你当前年龄段应该去适应的生活圈。开始有一些经历了，我们就容易回忆过去。会回忆孩童时代玩的游戏，然后纳闷现在孩子怎么都不在玩这些了；回忆当初追的歌星，现在怎么不红了；回忆当初读书时代玩的那么好的同学，怎么走着走着就散了，再也没有联系……

然而，每个人的一生本来就是一条独特的轨迹线，也正是因为它的独特，在生命的旅途中你才会走散很多朋友。正如泰戈尔所言，“天空不留下鸟的痕迹，但我已经飞过”。即便这一路走来你丢失了很多美好，但它们都是你生命中的弥足珍贵的精神财富，驱使着你不断向前。最后，让我们干了这碗鸡汤：向后看，人生是碎片的回忆，向前看，人生是无限的可能！
